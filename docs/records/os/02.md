### **Aim**
> To study basic Linux terminal commands for system navigation and directory management.


### **Commands**

- **`pwd`** (Print Working Directory):  
  Displays the absolute path of the current working directory.  
  *Syntax:* `pwd`

- **`ls`** (List):  
  Lists the files and directories present in the current or specified directory.  
  *Syntax:* `ls [options] [directory_name]`

- **`cd`** (Change Directory):  
  Used to navigate between different directories.  
  *Syntax:* `cd [path_to_directory]`

- **`mkdir`** (Make Directory):  
  Creates a new empty directory.  
  *Syntax:* `mkdir [directory_name]`

- **`rmdir`** (Remove Directory):  
  Deletes an empty directory.  
  *Syntax:* `rmdir [directory_name]`


### **Usage**

```bash
$ pwd
/home/user

$ mkdir linux_lab

$ ls
Desktop  Documents  Downloads  linux_lab  Music  Pictures

$ cd linux_lab
$ pwd
/home/user/linux_lab

$ cd ..
$ pwd
/home/user

$ rmdir linux_lab
$ ls
Desktop  Documents  Downloads  Music  Pictures

```

### **Conclusion**
> The basic Linux terminal commands for system navigation and directory management (`pwd`, `ls`, `cd`, `mkdir`, `rmdir`) were successfully executed, and their functions were practically understood.

### **Aim**
> To perform text processing, data filtering, and input/output redirection in Linux.


### **Commands**

- **`grep`**:  
  Searches for a specific string or pattern in a file  
  *Syntax:* `grep "[pattern]" [filename]`

- **`awk`**:  
  A pattern scanning and processing language, often used to extract specific columns  
  *Syntax:* `awk '{print $1}' [filename]`

- **`sed`** (Stream Editor):  
  Used for filtering and transforming text, like finding and replacing words  
  *Syntax:* `sed 's/[old_word]/[new_word]/g' [filename]`

- **`cut`**:  
  Removes specific sections or columns from each line of a file  
  *Syntax:* `cut -d'[delimiter]' -f[field_number] [filename]`

- **`sort`**:  
  Sorts lines of text files alphabetically or numerically  
  *Syntax:* `sort [filename]`

- **`uniq`**:  
  Filters out adjacent repeated lines in a file  
  *Syntax:* `uniq [filename]`

- **`diff`**:  
  Compares two files line by line and highlights the differences  
  *Syntax:* `diff [file1] [file2]`

- **Redirection (`>`, `>>`) & Piping (`|`)**:  
  `>` overwrites output to a file, `>>` appends to a file, and `|` passes the output of one command as the input to the next  
  *Syntax:* `command > [file]` or `command1 | command2`


### **Usage**

```bash
$ echo -e "apple\nbanana\napple\norange\ngrape" > fruits.txt

$ sort fruits.txt
apple
apple
banana
grape
orange

$ sort fruits.txt | uniq
apple
banana
grape
orange

$ grep "banana" fruits.txt
banana

$ sed 's/apple/mango/g' fruits.txt > new_fruits.txt

$ diff fruits.txt new_fruits.txt
1c1
< apple
---
> mango
3c3
< apple
---
> mango

```


### **Conclusion**
> Various text processing tools (`grep`, `sed`, `sort`, `uniq`, `diff`) and output redirection concepts (`>`, `|`) were successfully tested and executed on text files.

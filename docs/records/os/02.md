### **Aim**
> To study and execute basic Linux terminal commands for system navigation and directory management.


### **Theory**

Linux is a Unix-like, open-source operating system. The shell (terminal) is a command-line interpreter that provides a text-based interface to interact with the OS. Navigating the file system and managing directories are the most fundamental skills required to work in a Linux environment. In Linux, everything is structured in a hierarchical directory tree starting from the root directory (`/`).

#### **Commands & Syntax:**
- **`pwd`** (Print Working Directory):  
  Displays the absolute path of the current working directory.  
  *Syntax:* `pwd`

- **`ls`** (List):  
  Lists the files and directories present in the current or specified directory.  
  *Syntax:* `ls [options] [directory_name]`

- **`cd`** (Change Directory):  
  Used to navigate between different directories.  
  *Syntax:* `cd [path_to_directory]`

- **`mkdir`** (Make Directory):  
  Creates a new empty directory.  
  *Syntax:* `mkdir [directory_name]`

- **`rmdir`** (Remove Directory):  
  Deletes an empty directory.  
  *Syntax:* `rmdir [directory_name]`


### **Commands Usage**

:::info

- Lines starting with `$` indicate terminal commands to be executed.
- Lines starting with `#` are comments or explanations and should not be executed in the terminal.

:::

```bash title="Checking the current directory"
$ pwd
/home/user

```

```bash title="Creating a new directory"
$ mkdir linux_lab

```

```bash title="Listing the contents to verify the directory was created"
$ ls
Desktop  Documents  Downloads  linux_lab  Music  Pictures

```

:::info
Optional: Use `ls -l` to show a detailed list view 
:::

```bash title="Navigating into the newly created directory"
$ cd linux_lab
$ pwd
/home/user/linux_lab

```

```bash title="Moving back to the parent directory"
$ cd ..
$ pwd
/home/user

```

```bash title="Removing the empty directory"
$ rmdir linux_lab
$ ls
Desktop  Documents  Downloads  Music  Pictures

```


### **Conclusion**
The basic Linux terminal commands for system navigation and directory management (`pwd`, `ls`, `cd`, `mkdir`, `rmdir`) were successfully executed, and their functions were practically understood.

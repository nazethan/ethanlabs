### **Aim**
> To study and implement various data types in SQL

### **Theory**
**Different datatypes:**
- **Numeric Data Types:**
  - `INT`: Integer values
  - `DECIMAL(p, s)`: Fixed-point numbers with precision `p` and scale `s`
  - `FLOAT`: Floating-point numbers

- **String Data Types:**
  - `CHAR(n)`: Fixed-length string of length `n`
  - `VARCHAR(n)`: Variable-length string with a maximum length of `n`
  - `TEXT`: Large text data
  - `BLOB`: Binary Large Object for storing binary data
  - `LONGBLOB`: Large Binary Large Object for storing very large binary data
  - `ENUM`: Enumeration, a string object with a predefined set of values

- **Date and Time Data Types:**
  - `DATE`: Date values (YYYY-MM-DD)
  - `TIME`: Time values (HH:MM:SS)
  - `YEAR`: Year values (YYYY)


### **Example Script**
```sql title='Implementation of datatypes'
CREATE TABLE ProductInventory (
    ProductID INT,
    Price DECIMAL(10, 2),
    Weight FLOAT,
    ProductCode CHAR(5),
    ProductName VARCHAR(100),
    Description TEXT,
    ProductImage BLOB,
    LargeImage LONGBLOB,
    Category ENUM('Electronics', 'Clothing', 'Furniture'),
    MfgDate DATE,
    MfgTime TIME,
    MfgYear YEAR
);
DESCRIBE ProductInventory;
```

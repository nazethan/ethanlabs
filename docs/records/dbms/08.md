### **Aim**
> To write SQL queries for aggregation functions in SQL

### **Basic Syntax**
```sql
SELECT aggregation_function(column_name)
FROM table_name
WHERE condition;
```

### **Example Script**
```sql title='Aggregation Functions Examples'
-- Create a sample table for demonstration
CREATE TABLE Sales (
    SaleID INT PRIMARY KEY,
    ProductName VARCHAR(50),
    Category VARCHAR(30),
    SalespersonID INT,
    SalespersonName VARCHAR(50),
    SaleAmount DECIMAL(10, 2),
    SaleDate DATE,
    Region VARCHAR(25)
);

-- Insert sample data
INSERT INTO Sales VALUES 
(1, 'Laptop', 'Electronics', 101, 'John Smith', 1200.00, '2024-01-15', 'North'),
(2, 'Mouse', 'Electronics', 102, 'Alice Johnson', 25.00, '2024-01-16', 'South'),
(3, 'Chair', 'Furniture', 101, 'John Smith', 150.00, '2024-01-17', 'North'),
(4, 'Keyboard', 'Electronics', 103, 'Bob Wilson', 75.00, '2024-01-18', 'East'),
(5, 'Desk', 'Furniture', 102, 'Alice Johnson', 300.00, '2024-01-19', 'South');

-- Display original data
SELECT * FROM Sales;

-- 1. COUNT() - Count total number of sales
SELECT COUNT(*) AS TotalSales FROM Sales;

-- 2. COUNT() - Count distinct products
SELECT COUNT(DISTINCT ProductName) AS UniqueProducts FROM Sales;

-- 3. SUM() - Total sales amount
SELECT SUM(SaleAmount) AS TotalRevenue FROM Sales;

-- 4. AVG() - Average sale amount
SELECT AVG(SaleAmount) AS AverageSale FROM Sales;

-- 5. MIN() and MAX() - Minimum and maximum sale amounts
SELECT 
    MIN(SaleAmount) AS LowestSale,
    MAX(SaleAmount) AS HighestSale 
FROM Sales;
```
### **Aggregation Results**
  - **Total Sales Count:** 10
  - **Unique Products:** 10
  - **Total Revenue:** 4100.00
  - **Average Sale:** 410.00
  - **Min/Max Sales:** Min: 25.00, Max: 1200.00

### **Aim**
> To demonstrate the use of various DML operations i.e., insertion, deletion, and updating

### **Theory**
:::note
Write the following:
- [INSERT INTO](https://www.w3schools.com/MySQL/mysql_insert.asp)
- [DELETE](https://www.w3schools.com/MySQL/mysql_delete.asp)
- [UPDATE](https://www.w3schools.com/MySQL/mysql_update.asp)
:::

### **Commands**
```sql
INSERT INTO Employees (EmployeeID, FirstName, LastName, Age, Department)
VALUES (1, 'John', 'Doe', 30, 'HR');
VALUES (2, 'Jane', 'Smith', 25, 'IT');

DELETE FROM Employees
WHERE EmployeeID = 1;

UPDATE Employees
SET Age = 31
WHERE EmployeeID = 2;
```

:::note Table
Draw the table that the following script will give as output!
```sql
DESCRIBE Employees;
```
:::

### **Aim**
> To write SQL queries for data insertion and deletion

### **Basic Syntax**
- **INSERT Command**
  ```sql
      INSERT INTO table_name (column1, column2, ...)
      VALUES (value1, value2, ...);
  ```
- **DELETE Command**
  ```sql
      DELETE FROM table_name
      WHERE condition;
  ```

### **Example Script**
```sql
CREATE TABLE Products (
    ProductID INT PRIMARY KEY,
    ProductName VARCHAR(50),
    Category VARCHAR(30),
    Price DECIMAL(10, 2)
);

INSERT INTO Products (ProductID, ProductName, Category, Price) 
VALUES (101, 'Laptop', 'Electronics', 45000.00);

INSERT INTO Products (ProductID, ProductName, Price) 
VALUES (102, 'Notebook', 50.00);

INSERT INTO Products (ProductID, ProductName, Category, Price)
VALUES 
(103, 'Smartphone', 'Electronics', 15000.00),
(104, 'Desk Lamp', 'Furniture', 1200.50),
(105, 'Office Chair', 'Furniture', 5500.00);

DESCRIBE Products;

SELECT * FROM Products;

DELETE FROM Products
WHERE ProductID = 102;

DELETE FROM Products
WHERE Category = 'Furniture';

DELETE FROM Products;
```

### **Conclusion**
> Successfully implemented Data Manipulation Language (DML) commands for insertion and deletion.

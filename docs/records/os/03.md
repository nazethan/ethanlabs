### **Aim**
> To study file reading operations and text editors in Linux


### **Commands**
- **`cat`** (Concatenate):  
  Reads data from the file and outputs its content to the terminal. It can also be used to create files.  
  *Syntax:* `cat [filename]` or `cat > [filename]`
  
- **`nano`**:  
  A simple, user-friendly, and modeless command-line text editor.  
  *Syntax:* `nano [filename]`


- **`vi`**:  
  A powerful, modal visual text editor standard on most Linux systems.  
  *Syntax:* `vi [filename]`


### **Usage**

```bash
$ cat > sample.txt
This is a test file created using cat. # output
# Press Ctrl+D to save and exit

$ cat sample.txt
This is a test file created using cat. # output

$ nano nano_file.txt
# The nano editor opens.
# Type some text, press Ctrl+O to save, and Ctrl+X to exit

$ vi vi_file.txt
# The vi editor opens.
# Press 'i' to enter Insert mode. Type some text. 
# Press 'Esc' to exit Insert mode. Type ':wq' and press Enter to save and quit)

```

> The commands for reading and creating files (`cat`) and editing text using command-line editors (`nano`, `vi`) were successfully executed
